üéØ Overview

Phase 8 introduces AI-powered portfolio analysis and DEX swap functionality to Mercurius Hub using React 19, Firebase, TanStack Query, and 1inch API.

  • AI Copilot: Chat interface with streaming responses for portfolio analysis

‚öôÔ∏è Services API Reference

AI Service

ASYNC analyzePortfolioWithAI(portfolioData, userQuestion)

Description: Analyzes portfolio data with AI and returns analysis + recommendations

Parameters

Name Type Description
portfolioData PortfolioData Object with assets array, totalValue, assetCount
userQuestion string User's question about portfolio

Returns

{
  analysis: string,          // AI-generated analysis
  recommendations: [
    {
      type: 'buy'|'sell'|'rebalance'|'diversify',
      asset: string,         // Asset symbol or 'portfolio'
      rationale: string,
      priority: 'high'|'medium'|'low',
      estimated_impact: number  // percentage -100 to 100
    }
  ]
}

Example

const analysis = await analyzePortfolioWithAI(
  {
    assets: [{symbol: 'BTC', quantity: 1, currentValue: 45000}],
    totalValue: 45000,
    assetCount: 1
  },
  'Devo rebalancear meu portf√≥lio?'
);
console.log(analysis.analysis);
analysis.recommendations.forEach(rec => console.log(rec));

Swap Service

ASYNC getSwapQuote(tokenIn, tokenOut, amountIn, testnet, slippage)

Description: Fetches swap quote from 1inch API

Parameters

Name Type Default
tokenIn string -
tokenOut string -
amountIn string -
testnet boolean true
slippage number 0.5
POST /api/swap/quote

Returns

{
  toAmount: string,          // Output amount in smallest units
  estimatedGas: string,      // Gas estimate
  data: string,              // Encoded swap data
  to: string,                // Router address
  value: string              // ETH value
}

Description: Submits signed transaction to blockchain

Returns

{
  txHash: string,            // Transaction hash
  status: 'pending'|'success'|'failed'
}

Hook Pattern with TanStack Query

Async Hook with Caching

import { useQuery } from '@tanstack/react-query';

export function useMyData(id) {
  const { data, isLoading, error } = useQuery({
    queryKey: ['mydata', id],
    queryFn: async () => {
      const response = await fetch(`/api/data/${id}`);
      return response.json();
    },
    staleTime: 30 * 1000, // 30 seconds
    gcTime: 5 * 60 * 1000, // 5 minutes
    retry: 2
  });

  return { data, isLoading, error };
}

Service Function Pattern

Pure Service with Zod Validation

import { z } from 'zod';

const ResponseSchema = z.object({
  status: z.string(),
  data: z.unknown()
});

export async function fetchData(params) {
  const response = await fetch('/api/endpoint', {
    method: 'POST',
    body: JSON.stringify(params)
  });

  const json = await response.json();
  const validated = ResponseSchema.parse(json);
  return validated;
}

Context Pattern

React Context with Provider

import React, { createContext, useContext, useState } from 'react';

const MyContext = createContext(null);

export function MyProvider({ children }) {
  const [state, setState] = useState(null);

  return (
    <MyContext.Provider value={{ state, setState }}>
      {children}
    </MyContext.Provider>
  );
}

export function useMyContext() {
  const context = useContext(MyContext);
  if (!context) throw new Error('Must use inside MyProvider');
  return context;
}

Service Layer Errors

API Error Handling

try {
  const quote = await getSwapQuote(tokenIn, tokenOut, amount);
} catch (error) {
  console.error('Quote fetch failed:', error);
  // Zod validation error
  if (error instanceof z.ZodError) {
    console.error('Schema validation failed:', error.errors);
  }
  // Network error
  if (error instanceof TypeError) {
    console.error('Network error:', error.message);
  }
  // Re-throw or handle gracefully
  throw new Error('Failed to fetch quote');
}
