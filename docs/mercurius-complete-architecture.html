üîÑ Data Flow Architecture

Data Flow Examples

Portfolio Update Flow

  1. User adds asset (Portfolio.jsx)
  2. Data validated with Zod schema
  3. Written to Firestore (users/{uid}/portfolio)
  4. usePortfolioContext updates state
  5. TanStack Query refetches prices
  6. Components re-render with new values
  7. Daily snapshot captured (optional)

Swap Execution Flow

  1. User selects tokens and amount (SwapWidget.jsx)
  2. useSwapQuote fetches 1inch quote (TanStack Query)
  3. Quote cached for 30 seconds
  4. User confirms and clicks execute
  5. MetaMask opens for transaction signing
  6. Signed transaction sent to blockchain
  7. User portfolio automatically updated
